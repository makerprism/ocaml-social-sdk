# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - Unreleased

### Added

- OAuth 1.0a authentication with HMAC-SHA1 signature generation
- Streaming API support (filter stream, sample stream)
- Collections API (create collections, add/remove tweets)
- Saved Searches API
- Chunked media upload (INIT, APPEND, FINALIZE, STATUS phases)
- oEmbed API for embeddable tweet HTML
- Geo API with reverse geocoding
- CPS-based architecture for runtime independence
